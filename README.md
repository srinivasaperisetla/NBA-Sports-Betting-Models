# NBA-Sports-Betting-Models

This project is an NBA sports betting prediction system with a FastAPI backend and a Streamlit frontend.  
The backend serves machine learning model predictions, while the frontend provides an interactive user interface.

---

## Installation

Make sure you are using **Python 3.9 or higher**.

Install all dependencies:

```bash
pip3 install -r requirements.txt

cd backend
python3 main.py

cd ..
streamlit run app.py
